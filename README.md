# cn-site-hub

## 项目简介

cn-site-hub 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为静态 HTML 文件的集中存储与展示空间。页面内容可能涵盖各类实用工具、信息展示或小型应用，具体用途请参考各页面内部说明。

## 目录说明

```
/
├── index.html          # 仓库首页，列出所有已归档页面
├── pages/              # 存放所有独立 HTML 页面文件的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源文件（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下每个文件对应一个独立的 HTML 页面。
- `assets/` 目录存放页面共享的静态资源，便于统一维护。

## 页面归档说明

- 所有页面均为纯前端 HTML 文件，不依赖后端服务。
- 每个页面应尽可能自包含，如需引用外部资源，建议使用相对路径指向 `assets/` 目录。
- 页面内容应遵守相关法律法规，不得包含违法或侵权信息。
- 归档的页面可能不定期更新或移除，恕不另行通知。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，但需确保内容合规且与现有页面不重复。
- 提交前请将页面文件放入 `pages/` 目录，并在 `index.html` 中添加相应链接。
- 若发现页面存在错误或失效链接，可通过 Issue 反馈。
- 本仓库由维护者根据自身判断决定是否合并变更。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 即可浏览所有页面列表。
3. 点击列表中的链接可跳转至对应页面。
